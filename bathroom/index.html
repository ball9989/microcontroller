Our Project

โครงงานนี้เป็นโครงการเกี่ยวกับการสร้างเซนเซอร์ (Sensor) สำหรับตรวจสอบห้องน้ำที่มีสถานะเข้าใช้งานได้

จากปัญหาในการใช้ห้องน้ำที่ใช้เวลาในการรอเพื่อเข้าใช้เป็นเวลานาน อีกทั้งไม่สามารถรับรู้ได้ว่าห้องน้ำมีสถานะพร้อมใช้งานหรือไม่

Purpose Of Our Project

อำนวยความสะดวกให้กับผู้ที่ต้องการใช้งาน โดยนำเทคโนโลยีสารสนเทศมาประยุกต์ใช้งานร่วมกับ Microcontroller และจอ LCD Module

แสดงผลสถานะของห้องน้ำให้ผู้ใช้งานรับรู้

  • เกิดความสะดวกรวดเร็ว
  • ลดความผิดพลาด
  • ง่ายต่อการดูแล

Applications of the project




To find the parking space

สามารถประยุกต์ใช้กับการแสดงสถานะที่จอดรถที่ว่างอยู่



To check mailbox

สามารถประยุกต์ใช้กับการแสดงสถานะที่ตู้รับจดหมาย



TO measures something

สามารถประยุกต์ใช้กับการวัดระยะทางหรือระดับต่างๆ เช่น ระดับน้ำ เป็นต้น

Tools

  • Arduino Uno R3

    คือ ไมโครคอนโทรลเลอร์บอร์ดแบบสำเร็จรูปในยุคปัจจุบัน สามารถนำไปพัฒนาต่อเป็นอุปกรณ์ต่างๆ ได้

  • HC-SR04 Ultrasonic Sensor

    คือ เซนเซอร์วัดระยะทาง โดยไม่ต้องมีการสัมผัสกับตำแหน่งที่ต้องการวัด วัดได้ตั้งแต่ 2 cm ถึง 400 cm

  • 1602 Module LCD display + I2C interface

    คือ จอเเสดงผลที่ได้เป็นตัวอักษร สามารถสื่อสารกับบอร์ด Arduino ได้โดยการใช้ขาสัญญาณเพียง 2 ขา

Library & void setup

arduino

การต่อจอ LCD จะต้องมีการโหลด libary ของ "wire.h"
และ "LiquidCrystal_I2C.h"
โดยต้องมีการบอก address ซึ่งทั่วไปจะมี 2 ตัว
คือ 0x27 และ 0x3F ซึ่งขึ้นอยู่กับแต่ละคอมพิวเตอร์

การต่อ sensor จะต้องมีการโหลด libary ของ "Ultrasonic.h"

เมื่อทำการต่อสายไฟกับ HC-SR04 ULTRASONIC SENSOR
จะมีการต่อแบบ input และแบบ output
